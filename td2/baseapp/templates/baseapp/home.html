How does it work?

You can just create a story . Stories are private by default, but you can make it public by checking the 'Show publically?' box when you create it.

NB: Contest Entries must have always been private in order to be entered

NBB: To hide all your stories, you can set your 'Highest Access' field in your Profile Settings to Private. Turning it on again will go back to the indivudual story settings

You can also enter contests. Currently you have to do this via the Enter Contest button on the Current Contests page using an existing private story or a new one entirely.

Contest Process

  1. Someone creates a contest
  2. While that contest is open, users can enter stories into it
  3. Currently there is no scheduling (because this site is on a free hosting account), so the contest can be closed by a button on the Current Contests page. Do this once there are more than three entrants because...
  4. All entrants are then assigned three stories to critique. Users can see stories they are in line to judge in Judgemode
  5. User/Judges critiques aren't final until they have submitted with the 'Final' checkbox checked, so they can take notes as they read each story
  6. Once all the critiques are in (current state viewable in Current Contests another button on that page closes the competition and shows the result. Or it closes automagically and results become available from the contests page if my most recent code change works.
